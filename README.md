# Digital Image Processing

Repository for the undergraduate course **Digital Image Processing**.  
This repository is intended to organize lecture notes, practical assignments, projects, experiments, and study materials related to the knowleage.

---

## Course Overview

Digital Image Processing is an undergraduate-level course focused on the fundamentals of manipulating digital images using computational techniques. It covers both theoretical concepts and practical methods widely applied in computer vision and image analysis.

The course introduces students to the main techniques used to process digital images, enhance visual information, extract meaningful features, and support tasks such as segmentation, representation, and classification.

---

## Objectives

Provide the fundamental knowledge required to manipulate digital images using computers, while presenting important application areas and the main techniques used in the field.

---

## Syllabus

The course covers the following topics:

- Fundamentals of image processing
- Image enhancement
- Gray-scale transformation techniques
- Image smoothing
- Edge sharpening
- Spatial filtering
- Image transforms
- Image segmentation
- Image representation and classification

---

## Course Content

Main topics addressed throughout the course include:

- Origin and goals of image processing
- Fundamentals of computer vision and its relationship with human vision
- Image acquisition devices and methods
- Sampling and quantization
- Color systems
- Gray-level transformation techniques
- Smoothing methods
- Edge sharpening
- Frequency-domain transforms, especially the Fourier Transform
- Image segmentation
- Image analysis
- Feature extraction
- Basic concepts of representation and classification
